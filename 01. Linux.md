## Какой способ выбрать?

| Способ                                | Для кого                                                                                    | Плюсы                                                                                    | Минусы                                                                                  |
| ------------------------------------- | ------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| 1. WSL  (Windows Subsystem for Linux) | Точно для тебя! Остальные пункты просто прочитать.                                          | Интеграция с Windows, доступ к файлам обеих систем, хорошая производительность           | Не полноценный графический интерфейс (без дополнительной настройки), только для Windows |
| 2. Виртуальная машина                 | Для новичков и тех кто не готов парится с установкой                                        | Без риска для основной системы, можно откатить ошибки, полноценный графический интерфейс | Нужно устанавливать гипервизор, ниже производительность, требует больше ресурсов        |
| 3. Вторая ОС                          | Для тех, кому нужна полная производительность Linux для работы, но иногда требуется Windows | Максимальная производительность, доступ ко всем ресурсам компьютера                      | Сложнее настройка, риск потери данных, нужно перезагружаться для смены ОС               |
| 4. Основная ОС                        | Для самых смелых                                                                            | Полное погружение, максимальная производительность                                       | Требует отказа от привычных Windows-программ, кривая обучения круче                     |

1. Установка WSL
	- Устанавливаем [Windows Terminal](https://apps.microsoft.com/detail/9n0dx20hk701?hl=ru-RU&gl=RU)
	- Устанавливаем [WSL](https://learn.microsoft.com/ru-ru/windows/wsl/install)
	- Пользуемся
2. Использовать виртуальную машину. (Предпочтительно)
	- Устанавливаем [гипервизор](https://remontka.pro/hyper-v-windows-10/)
	- Скачиваем образ с [официального сайта](https://ubuntu.com/download/desktop) 
	- Создаём виртуальную машину
	- Пользуемся
3. Установить второй операционной системой (далее ОС) рядом с Windows. (Если у тебя один диск лучше выбери другой вариант).
	- Скачиваем образ с [официального сайта](https://ubuntu.com/download/desktop) 
	- Создаём [загрузочную флешку](https://internet-lab.ru/ubuntu_bootable_usb_on_windows)
	- Устанавливаем linux (гайдов огромное количество, если найдешь хороший, скинь я добавлю)
	- Пользуемся 
4. Установить linux как основную систему и пользоваться им постоянно. 
	- Скачиваем образ с [официального сайта](https://ubuntu.com/download/desktop) 
	- Создаём [загрузочную флешку](https://internet-lab.ru/ubuntu_bootable_usb_on_windows)
	- Устанавливаем linux (тоже самое что и пункт 3 только не размечаем диск)
	- Пользуемся

## 1. Знакомство с командной строкой (Терминал)

Независимо от способа, после первого входа выполни в терминале:

```bash
sudo apt update && sudo apt upgrade -y
```

**Что это значит:**
- `sudo` — выполнить команду с правами администратора (у тебя запросят пароль)
- `apt` — менеджер пакетов Ubuntu
- `update` — обновить информацию о доступных программах
- `upgrade` — установить обновления
- `-y` — автоматически отвечать "да" на вопросы
- `&&` — выполнить вторую команду, если первая завершилась успешно
***
Терминал (или командная строка) — это текстовый интерфейс для управления компьютером. Через него можно делать всё то же, что и через графический интерфейс, но быстрее, точнее и автоматизируемо.

**Как открыть терминал в Ubuntu:**
- Горячие клавиши: `Ctrl + Alt + T`
- Или найди "Терминал" в меню приложений

Гугление занимает почти половину всего рабочего времени в работе. Поэтому начнем потихоньку учиться искать информацию из открытых источников. 

Нужно научиться:
- Открывать терминал
- Переходить между директориями
- Смотреть содержимое директории (скрытые файлы тоже)
- Узнать рабочую директорию
- Узнать текущего пользователя
- Вывести содержимое файла на экран
- Создать/скопировать/переместить/удалить директорию
- Создать/скопировать/переместить/удалить файл
- Отредактировать созданный файл (например написать туда "hello" и сохранить)

Список команд, с которыми нужно познакомиться

```bash
cd 
ls
pwd
whoami
cat
cp
mv
rm
touch
nano
```
